# Security Policy

## Reporting Security Issues

Please do not open public issues for security problems involving authentication, API keys, private feeds, deployment credentials, or access controls.

Email: info@projectseven.us

Include:

- A short description of the issue
- Steps to reproduce, if safe to share
- Affected URL, endpoint, or component
- Any logs or screenshots with secrets removed

## Sensitive Data

Do not submit pull requests or issues containing:

- API keys
- Magic-link tokens
- Private deployment URLs
- Private feed credentials
- User email lists
- Audio that should not be public

## Scope

SignalForge is the public community and downstream surface. Core P7 Scanner security issues may be tracked privately first, then summarized publicly when safe.
